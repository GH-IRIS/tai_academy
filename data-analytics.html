Course Curriculum

A comprehensive, placement-aligned syllabus divided into logical phases.

Master spreadsheet systems. Build data cleaning protocols, calculate complex formulas, and structure basic dashboards.

  • Excel lookup structures (VLOOKUP, HLOOKUP, XLOOKUP)
  • Pivot tables, custom charts, descriptive formatting
  • Data cleaning, duplicate filtration, handling missing values
  • Logical formulas (IF, AND, OR) and string operations

Learn to interact with corporate data lakes using SQL queries. Write complex joins, subqueries, and grouping logic.

  • Relational database concepts & SQL syntax constructs
  • Data queries using SELECT, filtering (WHERE, HAVING)
  • Table relationships, Inner joins, Left/Right joins, Unions
  • Subqueries, nested queries, aggregation functions (SUM, AVG)

Learn to design executive dashboards. Map relationships, write DAX calculations, and publish KPI charts.

  • Power Query data importing, cleaning, transformations
  • Relationship modeling and schema configurations
  • DAX formulas (Calculated columns, Measures)
  • Building interactive reports, KPI tracking cards, maps

Write data scripts in Python. Clean large sets using Pandas, visualize trends, and model basic predictive AI regressions.

  • Python scripting basics, Notebook structures
  • Pandas dataframes, series, indexing, NumPy matrices
  • Data visualization with Matplotlib and Seaborn libraries
  • Intro to Predictive modeling, regressions, and forecasting
